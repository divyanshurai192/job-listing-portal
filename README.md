# Job Listing Portal (MERN Stack)

A full-stack Job Listing Portal connecting job seekers and employers.

## Features
- User Authentication (Admin, Employer, Job Seeker)
- Job Search & Filters
- Job Applications
- Admin Dashboard & Control Panel
- Secure JWT Authentication

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## How to Run
1. Start MongoDB
2. Backend:
   npm install
   node server.js
3. Frontend:
   npm install
   npm start
